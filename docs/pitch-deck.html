// The Education Engine

We move the lesson to the moment.

When

Lesson hits in the working-memory window.

The student sees the failure seconds after writing the line — not five days and four reviewers later. The mental context is still loaded.Mark · UC Irvine · 23-min refocus cost when learning is delayed

What

Concepts. Not solutions.

Three multiple-choice questions probe why the code fell — not what to type. The AI never writes the fix. The student must reason their way back to safe code.Retrieval practice · Productive struggle

How

Trial → Learn → Re-trial.

Pass the trial on your own. Re-run npx colosseum to confirm. You don't ship until you've proven the lesson stuck.Deliberate practice · Immediate feedback loop

// How We Used Kiro

Three layers of Kiro.

01

MCP Server

colosseum-mcp exposes custom tools — create_shadow_clone, release_gladiators, score_arena. Kiro summons the gladiators like any other MCP.

02

Agent Hooks

A pre-push hook intercepts risky code before it leaves the machine. Trial fires automatically — no extra command needed.

03

Spec-Driven Dev

Every gladiator, every guardrail, every honesty pill in the UI traces back to ARCHITECTURE_SPEC.md. Kiro built the system the spec described.
